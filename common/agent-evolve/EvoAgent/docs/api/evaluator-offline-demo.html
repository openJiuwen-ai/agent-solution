接口说明

    

      
提交

      
POST /evaluate/dataset · multipart：file（数据集）+ config（JSON blob）

      
config

      
{id_field, llm_config?, groups:[{name, kind, pred_field, gold_field, keywords, json_key, labels, extract_key, batch_metrics}]}

      
kind 条件显隐

      
exact_match → 必填 gold_field；keyword → 必填 keywords≥1；llm_judge → 必填 gold_field + labels（声明规范标签，≥1，禁止"其他"） + extract_key（需从 pred 中提取的内容，必填） + 顶层 llm_config（LLM 凭 extract_key 从 labels 选规范标签，不读 gold）。json_key 非空时 pred 列按 JSON 解析按键取值。gold 列 distinct 值必须 ⊆ labels。

      
batch_metrics 合法集合

      
["mean","precision","recall","f1","accuracy"]，多选各勾各返回。

      
查询

      
GET /evaluate/dataset/jobs/{job_id} 轮询；.../stream 订阅 SSE（支持 Last-Event-ID 重放）。

      
overall（跨组）

      
_overall（各组 composite 均值宏平均）+ P/R/F1/Acc（各组对应项宏平均）。混 keyword 组时 precision 被 1.0 抬高；llm_judge 组按真实 (gold, judged_label) 多类混淆，非退化，另出 其他_count/其他_rate 诊断键。

      
422 校验

      
未知 batch_metric / 空 groups / keyword 空 keywords / exact_match·llm_judge 缺 gold_field / llm_judge 缺 labels / llm_judge 声明"其他" / llm_judge 缺 extract_key / llm_judge gold 出现未声明值 / llm_judge 缺 llm_config。
