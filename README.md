## 项目结构

```
.
|-- common
|   |-- agent-runtime-ext-java
|   `-- example
|       |-- agentcore-ext-deepagent-remote-a2a-demo
|       |-- agentcore-ext-remote-a2a-tool-demo
|       `-- versatile-a2a-adapter-demo
|-- README.md
|-- LICENSE
```

- `common`：通用扩展模块和配套示例。
  - `agent-runtime-ext-java`：运行时扩展模块的 Maven 父工程，包含 adapter 扩展实现。
  - `example`：扩展模块对应的示例工程，包含 `agentcore-ext-deepagent-remote-a2a-demo`、`agentcore-ext-remote-a2a-tool-demo` 和 `versatile-a2a-adapter-demo`。
