### 5. 发现与说明

| 序号 | 类型 | 描述 | 处理 |
| --- | --- | --- | --- |
| 1 | 分层 | health / 含 LLM 单次对话 / 纯 API 热更 不宜都叫「集成」或都叫「E2E」 | 本报告按冒烟/集成/系统/E2E 重分级 |
| 2 | 语义 | restore 还原首次 update 快照 | 集成用例先 restore 再取基线 |
| 3 | 约束 | 提问携带预期标记属于作弊 | E2E 提问不含标记值 |
| 4 | 拓扑 | 多 ready sandbox 时 list_ready 脆弱 | 推荐 from_logs |
