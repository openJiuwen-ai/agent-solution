# 版本变更记录

本文档记录 EDPAgent 各版本的变更内容。

## v0.2.0-SNAPSHOT（当前开发版本）

**发布日期**：2026年7月

### 新特性

- ✅ **Rail 拦截器链**：9个 Rail 构成处理链（Cancel → Todo → ExecLimit → Mcp → Versatile → AskUser → Log → Event → Scripts）
- ✅ **Todo 状态机**：PENDING → IN_PROGRESS → COMPLETED/CANCELLED，支持动态路径调整
- ✅ **三层 Governance 配置体系**：
  - `planrule.yaml`：角色定义、职责边界、ReAct 协议、Skill 路由
  - `actrule.yaml`：执行约束、工具白名单、调用次数限制
  - `scriptconfig.yaml`：通用话术、思维链话术、ask_user 模板
- ✅ **场景化配置**：通过 `EDP_AGENT_SCENARIO_HOME` 环境变量一键切换业务场景，配置增量覆盖
- ✅ **Skill 技能系统**：
  - 声明式 YAML 定义（SKILL.yaml + SKILL.md + Python 脚本）
  - 三种加载模式：all / auto_list / none
  - 触发词自动路由
- ✅ **核心内置工具**：
  - `call_versatile`：Versatile 工作流委托（REST/A2A 双模式，支持中断续传）
  - `call_mcp`：MCP SSE 脚本沙箱调用（主备切换）
  - `ask_user`：用户追问与确认（中断机制）
  - `cancel_task`：任务取消
  - `todo_create/modify/list/get`：任务状态管理
- ✅ **工具间数据直通**：ToolDataChannel 机制，前序工具结果自动写入供后续读取
- ✅ **思维链话术**：real_stream / fixed_script 双模式，支持逐字渲染
- ✅ **A2A 协议通信**：JSON-RPC over SSE，标准 Agent Card 发现
- ✅ **Redis 状态存储**：Todo 和 Checkpoint 持久化，支持中断续传
- ✅ **Docker 部署**：基于 eclipse-temurin:21-jre-jammy，内置 Python 3 运行环境
- ✅ **Docker HEALTHCHECK**：通过 Agent Card 端点自动健康检查

### 内置场景

- **wealth-demo**：理财购买全流程演示场景
  - 4个 Skill：product_recommend、product_select、fund_planning、interact_finance_rec
  - 业务范围：理财产品推荐、筛选、购买、余额查询、转账、资金筹划
- **hz-zhidaitong**：智贷通场景（基础框架）

### 技术栈

- Spring Boot 3.x + Java 21
- DeepAgent / OpenJiuwen ReAct 引擎
- Redis（Spring Data Redis）
- Python 3（MCP 脚本运行时）
- SLF4J + Logback 日志
- Maven 构建

### 配置项

支持环境变量配置（约30个配置项），参见 [环境变量完整参考](../reference/环境变量参考.md)。

## 升级指南

### 从历史版本升级

当前为初始开发版本，暂无历史升级路径。后续版本升级指南将在此处提供。

### 升级注意事项

1. **备份配置**：升级前备份 `.env` 文件和自定义场景目录
2. **清理 Redis**：跨大版本升级建议清理 Redis 中的旧数据（Todo/Checkpoint），避免格式不兼容
3. **查看 changelog**：检查是否有配置项变更或废弃
4. **先测试验证**：在测试环境验证后再升级生产
