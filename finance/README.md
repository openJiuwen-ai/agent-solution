# finance — 金融行业 Agent 解决方案

本目录为 **agent-solution_finance** 的核心资产区，按能力类型划分为三个子目录：

| 目录 | 说明 |
|------|------|
| [methodology/](methodology/) | 方法论与流程工具——分析框架、模板、辅助脚本等 |
| [skills/](skills/) | 精品 Skill——面向 Agent 的可执行任务能力 |
| [mcp/](mcp/) | 精品 MCP——对接外部系统与数据源的服务实现 |

三者协同：`methodology` 定义规范与流程，`skills` 封装 Agent 能力，`mcp` 提供外部连接与数据访问。
