# PLAN — 加固项清单 & 未加固项风险说明

> v6 版本 | 2026-05-14

---

## 加固项总览

| ID | 加固项 | 等级 | 容器模式 | 说明 |
|----|--------|------|----------|------|
| H-01 | SSH 安全配置 | L2 | ⏭️ 跳过 | 禁用 root 登录、限制认证方式 |
| H-02 | OpenClaw Token 轮换 | L2 | ✅ 始终 | 生成新的 API Token |
| H-03 | OpenClaw 配置加固 | L3 | ✅ 始终 | 最小化配置、禁用危险功能 |
| H-04 | 文件权限修复 | L2 | ✅ 始终 | 修正关键文件权限 |
| H-05 | 系统内核参数 | L3 | ⚠️ 受限 | sysctl 安全调优 |
| H-06 | 防火墙规则 | L3 | ⏭️ 跳过 | iptables/nftables 限制 |
| H-07 | 文件完整性基线 | L2 | ✅ 始终 | 关键文件 SHA256 哈希 |
| H-08 | 容器安全配置 | L3 | ✅ 容器 | 只读根文件系统、seccomp |
| H-09 | 日志审计配置 | L2 | ✅ 始终 | 审计日志轮转和保留 |
| H-10 | 密码策略 | L2 | ⏭️ 跳过 | PAM 密码复杂度 |

---

## 各等级包含的加固项

### L1 — 基础加固
- H-02: OpenClaw Token 轮换
- H-04: 文件权限修复
- H-07: 文件完整性基线

### L2 — 标准加固
- L1 全部 +
- H-01: SSH 安全配置
- H-03: OpenClaw 配置加固
- H-09: 日志审计配置

### L3 — 增强加固（推荐）
- L2 全部 +
- H-05: 系统内核参数
- H-06: 防火墙规则
- H-08: 容器安全配置

### L4 — 严格加固
- L3 全部 +
- H-10: 密码策略
- 额外的审计日志配置
- 网络命名空间隔离

### L5 — 极限加固
- L4 全部 +
- SELinux 强制模式
- 最严格的 seccomp 策略
- 禁用所有非必要端口

## 未加固项风险说明

| 未加固项 | 风险描述 | 风险等级 | 建议 |
|----------|----------|----------|------|
| SSH 默认配置 | 暴力攻击、弱密码登录 | 🔴 高 | 至少禁用 root 登录 |
| OpenClaw 默认 Token | 未授权 API 访问 | 🔴 高 | 必须轮换 |
| 宽松文件权限 | 配置泄露、篡改 | 🟡 中 | 必须修复 |
| 无完整性监控 | 入侵后无法察觉 | 🟡 中 | 建议建立基线 |
| 无防火墙规则 | 未授权网络访问 | 🟡 中 | 建议限制 |
| 无内核调优 | 网络攻击面大 | 🟢 低 | 建议调优 |
| 无密码策略 | 弱密码账户 | 🟢 低 | 建议配置 |

## 执行顺序

```
预评估 (audit)
    ↓
加固 (harden) — 按等级包含的项依次执行
    ↓
后验证 (post-validate) — 验证加固效果 + Gateway 状态
    ↓
对比报告 (pre-post-diff) — 生成前后对比
    ↓
最终报告 (report) — 评分 + 状态矩阵
    ↓
[可选] 恢复 (restore) — 如果加固导致问题
    ↓
[可选] 恢复验证 (restore-validate) — 确认恢复成功
```

## 容器 vs 宿主机模式

脚本自动检测运行环境：

| 检查项 | 容器内 | 宿主机 |
|--------|--------|--------|
| systemd | ❌ 跳过 | ✅ 执行 |
| SELinux | ❌ 跳过 | ✅ 执行 |
| firewalld | ❌ 跳过 | ✅ 执行 |
| sysctl | ⚠️ 部分受限 | ✅ 执行 |
| SSH 配置 | ⚠️ 通常无 | ✅ 执行 |
| 文件权限 | ✅ 执行 | ✅ 执行 |
| 包管理 | ✅ apt/dnf | ✅ dnf/apt |
